# Architecture Design Examples

This directory contains AWS architecture design examples created to demonstrate solution architecture thinking across different use cases.

## Included Designs

### 1. Simple Web Application
Basic single-AZ architecture focused on low cost and quick deployment.

### 2. Highly Available Web Application
Multi-AZ production-ready design focused on availability, fault tolerance, and scaling.

### 3. Secure VPC Design
Security-first VPC architecture with private subnets, controlled access, and reduced attack surface.

## How to Read These Designs
Each design explains:
- The use case
- Key architectural decisions
- Trade-offs involved
- When the design is appropriate
